# Projeto de Hardware – Quiz Eletrônico com ESP32

## Visão Geral
Este projeto consiste no desenvolvimento de uma **placa eletrônica dedicada a um sistema de quiz**, integrando um microcontrolador ESP32, um display OLED via SPI e um alto-falante para feedback sonoro. O foco do projeto é a **integração elétrica e o layout da PCB**, garantindo funcionamento confiável e montagem prática.

## Componentes Principais
- **ESP32**: Microcontrolador central da placa, responsável pelo controle dos periféricos.
- **Display OLED SPI**: Interface visual para exibição das perguntas, respostas e estados do jogo.
- **Speaker**: Utilizado para feedback sonoro, como confirmação de respostas ou alertas.
- **Circuito de Alimentação**: Responsável por fornecer tensão estável aos componentes da placa.
- **Componentes Passivos**: Resistores, capacitores e demais elementos necessários para polarização, filtragem e funcionamento adequado do circuito.

## Projeto da PCB
- Layout desenvolvido visando **simplicidade, clareza e facilidade de montagem**.
- Trilhas dimensionadas de acordo com as correntes envolvidas no projeto.
- Separação lógica dos blocos da placa (controle, display e áudio) para melhor organização.
- Posicionamento dos componentes pensado para facilitar soldagem manual e testes.
- Uso de planos de alimentação e terra para reduzir ruídos e melhorar a estabilidade elétrica.

## Objetivo do Hardware
A placa foi projetada para servir como uma **base dedicada e confiável** para o sistema de quiz, permitindo integração direta dos periféricos sem a necessidade de módulos externos, resultando em um hardware compacto e funcional.
